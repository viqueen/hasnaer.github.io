Experimenting with ANTLR

by Hasnae Rehioui

  Posted on Friday Oct. 31st 2014

Being a person with interest in "anything" recognition (yes I do have some background in both voice and music recognition), it only made sense I play around with yet another tool for language recognition  ANTLR

As it happens, I once wrote my own arithmetic expression evaluator (CS blabla course), everything from parsing tokens into a tree, to rendering the result. I was so proud but needless to say that each time I had an epiphany  oh let's add some special vars into the expression tree and do some black magic woman ! (music pun booya)  I also had to dig into the rusty code and pray the lords I didn't introduce any "feature"

So now I decided to try out ANTLR4, turns out it's pretty easy to set up and use; which means I can finally unlaze* and implement some actual features I have in mind for a while
